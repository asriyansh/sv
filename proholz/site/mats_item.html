Финский клееный брус

Финские материалы

Клееный брус - лучший материал для строительства современного деревянного дома. Клееный профилированный брус - сочетание современных технологий с традиционными формами. Клееный брус позволяет избежать многих трудностей, связанных с использованием материалов из массива дерева.


Технология производства клееного бруса

Клееный брус производят методом склеивание ламелей. Количество ламелей выбирают в зависимости от сечения бруса. Сейчас широкое распространение получило склеивание ламелей из разных пород дерева, так для наружных ламелей можно использовать древесину из кедра или лиственницы, а внутри оставить сосну.

Для склеивания ламелей используют специальные пресса. Клей экологически чистый, не вызывающий аллергичиских реакций


Профили клееного бруса


Высота — от 169 до 240мм, толщина — от 70 до 240мм, D-профиль

Дома из клееного бруса можно разделить на три категории, в зависимости от категории выбрать сечение клееного бруса

Такое разделение, прежде всего, сделано для экономии ваших средств. Так толстые стены в доме для постоянного проживания помогут сэкономить деньги на отоплении, а тонкие стены в летнем домике существенно сократят кубатуру стен, по-этому стоимость будет ниже.

При необходимости мы сможем сделать дополнительное утепление стен. Для этого изготавливается вентилируемый фасад, утепленный базальтовым утеплителем и обшитый имитацией бруса.


Преимущества домов из клееного бруса

  • Кратчайшие сроки возведения стенового комплеaкта
  • Возможность выполнения больших пролетов без дополнительных точек опор
  • Лучшая теплоизоляция стен из клееного бруса, по сравнению со стенами из кирпича
  • Не дает усадку
  • Не растрескивается
  • Клееный брус натуральный дышащий материал
  • Экономия затрат на фундаментные работы, за счет меньшего веса стен из клееного бруса по сравнению со стенами из кирпича
